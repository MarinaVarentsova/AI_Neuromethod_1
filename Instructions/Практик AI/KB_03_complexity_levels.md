# KB_03_complexity_levels

## Назначение
Файл описывает уровни сложности практических заданий.

## Базовый уровень
Есть понятная ситуация, полные вводные, один основной алгоритм, минимум отвлекающих факторов.

Проверяет:
- узнавание типовой ситуации;
- применение базового алгоритма;
- выполнение действия по образцу;
- понимание простых критериев качества.

## Средний уровень
Есть неполные данные, несколько вариантов действия, нужно выбрать подходящий способ и объяснить решение.

Проверяет:
- анализ вводных;
- выбор решения;
- работу с ограничениями;
- обоснование действия;
- предотвращение типичной ошибки.

## Продвинутый уровень
Есть неоднозначность, конфликт целей, ограничения, риск ошибки, исключения или необходимость приоритизации.

Проверяет:
- принятие решений в сложной ситуации;
- работу с исключениями;
- оценку рисков;
- адаптацию алгоритма;
- перенос навыка в реальный рабочий контекст.

## Как повышать сложность
Можно усложнять задание через:
- неполные данные;
- лишние данные;
- ограничение времени;
- конфликт интересов;
- неопределённость;
- риск последствий;
- несколько заинтересованных сторон;
- необходимость выбрать между плохими вариантами;
- требование объяснить логику решения.
